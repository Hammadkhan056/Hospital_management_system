🏥 Hospital Management System (CLI-Based)
📘 Project Purpose

This project is created to practice Python programming, logic building, and problem solving.
It’s a command-line-based hospital management system that helps understand how real-world systems work step by step.

⚙️ Main Features

Add, view, and search patients
Add and view doctors
Book and view appointments
Simple menu-driven CLI interface
Step-by-step upgrade plan (OOP → File Handling → DSA → UI)

🚀 Learning Goals

Strengthen logic building and problem-solving skills
Practice data structures in a real-world context
Learn system design thinking from beginner to advanced level
